PRODUCTION STANDARDS:
- Selective Preservation: Preserve every part of the original image that is not explicitly specified for modification. Unspecified regions must remain visually identical to the original.
- Seamless Integration: All new or modified elements must integrate seamlessly into the existing image. Edges, shadows, reflections, and highlights at the boundary of any edit must match and blend naturally with the surrounding image content.
- Lighting Consistency: New and modified elements must share the same light source direction, color temperature, shadow softness, highlight intensity, and ambient light characteristics as the rest of the image. No element should appear as though it was lit differently.
- Color Harmony: Colors of all new elements must be fully harmonious with the existing color palette established in the project context and visible in the provided reference images. Use the established palette — do not introduce discordant new colors.
- Texture and Detail Parity: The texture resolution, surface detail, and rendering quality of all edited or added regions must precisely match the quality level of the unedited portions of the image. Edited areas must not appear more polished or less detailed than their surroundings.
- Line Work Consistency: If the art style uses outlines or ink lines, all new elements must use identical line weight, color, and stylistic treatment as the existing lines in the image.
- Character Accuracy: Any character-related edits must remain fully consistent with all character descriptions provided in the project context, including facial features, proportions, color palette, and distinctive identifiers.
- Zero Artifacts: The final output must be completely free of visible seams, blurring at edit boundaries, color fringing, smearing, inconsistent noise, or any other visual artifact that would reveal the image as having been edited.
- Production Readiness: The output must be ready for direct use in the game without any additional post-processing or cleanup.
